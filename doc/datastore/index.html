data store

Welcome to the data store module!!

You will find here access to various data store including relational, documented oriented, key/value, etc. But also, several mapping tools and/or utilities or data store extensions.

Let’s see what modules are available:

jdbc

The jdbc module provides a high performance connection pool for jdbc via Hikari.

mapping tools

  • ebean: object relational mapping via Ebean ORM.

  • hibernate: object relational mapping via Hibernate.

  • hibernate validator: bean validation.

  • jdbi: is fluent and lightweight SQL library for jdbc.

  • jOOQ: lets you build type safe SQL queries through its fluent API.

  • querydsl: unified Queries for SQL. Querydsl is compact, safe and easy to learn.

  • querydsl-jpa: unified Queries for JPA. Querydsl is compact, safe and easy to learn.

  • rxjdbc: efficient execution, concise code, and functional composition of database calls using JDBC and RxJava Observable

migrations

  • flyway: provides database migrations via Flyway.

mongodb

The mongodb provides access to MongoDB instances.

mapping tools

  • jongo: Query in Java as in Mongo Shell.

  • morphia: object to document mapper via Morphia.

async / reactive

  • mongodb-rx: provides composable asynchronous and event-based observable sequences for MongoDB.
