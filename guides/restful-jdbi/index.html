welcome

In this guide you will learn how to build a rest API for Pets and persist them in a relational database.

The services will be available at:

GET http://localhost:8080/pets (produces a **JSON** response with all the pets)

GET http://localhost:8080/pets/:id (produces a **JSON** with a single pet or 404 if not found)

POST http://localhost:8080/pets (require body as **JSON** and respond with the recently created pet)

PUT http://localhost:8080/pets (require body as **JSON** and respond with the updated pet)

requirements

Make sure you have all these software installed it in your computer:

  • A text editor or IDE
  • JDK 8+ or later
  • Maven 3+
  • Git

ready

ready

Open a terminal (console for Windows users) and paste:

git clone https://github.com/jooby-guides/restful-jdbi.git

cd restful-jdbi

An almost empty application is ready to run, you can try now with:

mvn jooby:run

Open a browser and type:

http://localhost:8080

TIP: If you are using an IDE that automatically compiles your source code while you save it… mvn jooby:run will detects those changes and restart the application for you!! more at mvn jooby:run.
